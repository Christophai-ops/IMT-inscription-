# 📌 Inscription / Association Membership App

## 📖 Project Description
This web application is designed for managing association registrations and member data collection.  
Users can register through a shareable link, and all data is stored in Firebase Firestore.

The system also supports offline mode, allowing users to submit data without internet connection.  
Once the internet is restored, the data is automatically synchronized with Firebase.

---

## 🚀 Key Features

- 📝 Online registration form for members
- 🌐 Shareable link (Facebook, Email, LinkedIn, etc.)
- 📊 Admin dashboard to view registered members
- 📡 Firebase Firestore database integration
- 📴 Offline mode (works without internet)
- 🔄 Automatic synchronization when back online
- 💾 Local storage backup (LocalForage)
- ⚡ Fast and lightweight web application

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (if used)
- JavaScript (Vanilla JS)
- Firebase Firestore
- Firebase CDN (no backend server required)
- LocalForage (offline data storage)
- Service Worker (PWA support)
